# Objection Triage Script

## Definition
Offering a menu of three root causes (price, timing, uncertainty it will work) forces the prospect to self-diagnose and name the real objection, which removes vagueness and puts the salesperson back in control of the conversation.

## When to Use


## Example Quote
> "Usually when people say let me think about it, it's usually the price, the timing, or you're not sure it will work. Which one is it?"
